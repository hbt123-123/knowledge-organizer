# 第1章 集合

## 1.1 集合的概念

**集合的定义**：集合是把具有共同特征的事物或对象汇总在一起的结果。集合中的每个对象称为元素。

例如：
- 自然数集合 N = {0, 1, 2, 3, ...}
- 整数集合 Z = {..., -2, -1, 0, 1, 2, ...}

## 1.2 集合的表示方法

### 列举法
将集合中的元素一一列举出来，用花括号括起来。

例如：A = {1, 2, 3, 4, 5}

### 描述法
用描述元素共同特征的方法表示集合。

例如：B = {x | x 是小于10的正偶数}

## 1.3 集合的基本关系

**子集**：如果集合A的所有元素都是集合B的元素，则称A是B的子集，记作 A ⊆ B。

**真子集**：如果 A ⊆ B 且 A ≠ B，则称A是B的真子集，记作 A ⊂ B。

**集合相等**：如果 A ⊆ B 且 B ⊆ A，则 A = B。

## 1.4 集合的运算

### 并集
A ∪ B = {x | x ∈ A 或 x ∈ B}

### 交集
A ∩ B = {x | x ∈ A 且 x ∈ B}

### 补集
设 U 为全集，A 的补集记作 ∁A = {x | x ∈ U 且 x ∉ A}

## 例题

**例1**：设 A = {1, 2, 3}，B = {2, 3, 4}，求 A ∪ B 和 A ∩ B。

解：
- A ∪ B = {1, 2, 3, 4}
- A ∩ B = {2, 3}

**例2**：设全集 U = {1, 2, 3, 4, 5}，A = {1, 2, 3}，求 ∁A。

解：∁A = {4, 5}
