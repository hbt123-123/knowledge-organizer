# 第2章 函数

## 2.1 函数的概念

**函数的定义**：设 A 和 B 是非空的数集，如果按照某种确定的对应关系 f，使对于集合 A 中的任意一个数 x，在集合 B 中都有唯一确定的数 f(x) 和它对应，那么就称 f: A → B 为从集合 A 到集合 B 的一个函数。

记作：y = f(x)，x ∈ A

其中：
- x 称为自变量
- y 称为因变量
- A 称为定义域
- {f(x) | x ∈ A} 称为值域

## 2.2 函数的表示方法

### 解析式法
用数学式子表示函数关系。

例如：f(x) = 2x + 1

### 图像法
用坐标系中的图像表示函数。

### 列表法
用表格形式列出函数值。

## 2.3 函数的性质

### 单调性
- 增函数：在区间内，x 增大时 f(x) 也增大
- 减函数：在区间内，x 增大时 f(x) 减小

### 奇偶性
- 奇函数：f(-x) = -f(x)，图像关于原点对称
- 偶函数：f(-x) = f(x)，图像关于 y 轴对称

### 周期性
若存在非零常数 T，使得 f(x + T) = f(x) 对定义域内所有 x 成立，则 T 称为函数的周期。

## 2.4 常见函数类型

### 一次函数
f(x) = kx + b (k ≠ 0)

### 二次函数
f(x) = ax² + bx + c (a ≠ 0)

顶点公式：x = -b/(2a)

### 反比例函数
f(x) = k/x (k ≠ 0)

## 例题

**例1**：求函数 f(x) = x² - 2x + 1 的顶点坐标。

解：
x = -(-2)/(2×1) = 1
f(1) = 1 - 2 + 1 = 0
顶点坐标为 (1, 0)

**例2**：判断函数 f(x) = x³ 的奇偶性。

解：
f(-x) = (-x)³ = -x³ = -f(x)
所以 f(x) = x³ 是奇函数。
